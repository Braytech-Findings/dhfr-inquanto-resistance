# What the Project Proves

This is an internal evidence guide. Any interpretation is a **DRAFT TECHNICAL
INTERPRETATION FOR RESEARCHER REVIEW**.

## Verified now

- One WT_TMP CAS(6,6)/12-qubit saved-parameter UCCSD workflow produced the
  protected ideal and finite-shot energies.
- The finite-shot run used 576 measurement circuits × 100 shots on the local
  noiseless H2-1LE emulator.
- The four compact molecular cluster input files are readable and pass basic
  coordinate/duplicate checks under a shared preparation description.
- The repository's local safety, provenance, statistical, backend, and routing
  tests pass without network access.

## Supported with limitations

- Local quantum-chemistry software can reconstruct and measure the saved WT_TMP
  model.
- Pilot classical interaction proxies exist for all four systems and support
  model-sensitivity exploration, but are not binding free energies.
- Candidate ligand-focused CAS(6,6) spaces pass basic CASCI occupation checks,
  but cross-system orbital identity is unresolved.

## Not yet proven

Matched four-system quantum effects, a resistance mechanism, ligand superiority,
clinical usefulness, hosted-emulator molecular execution, hardware execution,
quantum advantage, and independent biological validation.

## Required next evidence

Resolve matched orbital correspondence; build and validate four comparable
Hamiltonians; optimize compatible parameters separately; reproduce ideal and
finite-shot outputs with seeds/checkpoints; complete prespecified sensitivity;
then, only after local review, perform an explicitly approved ten-shot Nexus
smoke test before considering molecular remote execution.
